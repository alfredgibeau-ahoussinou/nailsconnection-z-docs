# E-mails (Resend)

Optionnel : sans configuration, la réservation fonctionne quand même (confirmation à l’écran).

## Variables Netlify (noms seulement)

| Variable | Rôle |
|----------|------|
| `RESEND_API_KEY` | Clé API Resend |
| `OWNER_EMAIL` | Boîte qui reçoit les alertes « nouvelle demande » |
| `EMAIL_FROM` | Expéditeur vérifié chez Resend |

Ne jamais publier les valeurs dans ce dépôt.

## Compte

Créer un compte sur [resend.com](https://resend.com), vérifier le domaine ou utiliser l’adresse de test fournie.
