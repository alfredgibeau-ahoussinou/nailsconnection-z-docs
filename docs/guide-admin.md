# Guide admin

Accès : bouton **`dev`** en bas de page → mot de passe (communiqué en privé).

## Onglets

1. **À traiter** — demandes en attente (accepter / refuser)
2. **Planning** — jour par jour, fermer / rouvrir un créneau
3. **Rendez-vous** — liste et historique des RDV
4. **Mon salon** — horaires, services, réglages

## Statuts

| Statut | Signification |
|--------|----------------|
| À confirmer | Demande reçue, créneau **non bloqué** |
| Confirmé | Accepté, créneau **réservé** |
| Refusé | Refusé |
| Annulé | Annulé |
| Terminé | Prestation réalisée |

## Règle importante

Seuls les rendez-vous **Confirmés** bloquent le créneau. Les demandes en attente ne ferment pas le planning.
