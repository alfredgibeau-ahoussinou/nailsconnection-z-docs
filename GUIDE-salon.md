# Guide salon — NailsConnection.Z

Ce guide regroupe ce qu’il faut savoir pour gérer le site au quotidien, **sans toucher au code**.

## 1. Par où commencer

1. Lis le [résumé des politiques](./docs/politiques-resume.md).
2. Ouvre l’**espace admin** via le bouton discret `dev` en bas du site (mot de passe communiqué **en privé** — jamais dans ce dépôt).
3. Consulte le [guide admin](./docs/guide-admin.md) pour traiter les demandes et le planning.
4. Paiement : **espèces uniquement**, acompte **10 €** avant la prestation.

## 2. Flux réservation

1. La cliente envoie une demande → statut **À confirmer** (`pending`).
2. Le créneau n’est **pas bloqué** tant que la demande n’est pas acceptée (plusieurs demandes peuvent viser le même créneau).
3. Tu **acceptes** ou **refuses** dans l’admin → le créneau est réservé seulement si **Confirmé**.
4. La cliente confirme la veille ; sans confirmation, le créneau peut être libéré.
5. Acompte 10 € en espèces + solde le jour J (espèces).

## 3. Horaires par défaut

Voir [docs/horaires.md](./docs/horaires.md). En résumé : mardi → vendredi 10h–19h, samedi 10h–18h, créneaux 30 min, délai minimum 24 h.
